ПРОВЕРКА АВТОРИЗАЦИИ. Реализуется с помощью app.use('path', checkAuth())

1) Реализация регистрации
2) Реализация авторизации. Получение id, имени и авы пользователя
3) POST /dialog/newUserID/
3) GET /dialogs/id - возращает список диалогов у текущего пользователя (с учетом пагинации). Каждый диалог имеет:
   А) id
   Б) информацию о собеседнике
   В) Последнее сообщение
4) GET /messages/dialogId (с учетом пагинации). Возращает массив сообщений (в том числе id). Каждое сообщение имеет статус, прочитано ли оно
5) Read /message/id
6) Delete /message/id
5) GET /user/online - возращает дату, когда пользователь последний раз появлялся в сети.
6) Subscribe at user on/offline - получаем по весокетам, когда юзверь отлогинится/залогиница

7) Сама личка
8) ... печатает
9) Загрузка и хранение фотачек. Лэйзилоад

1) Создание диалогов между собеседниками
   А) Хранятся все сообщения
   Б) Хранится id собеседника
2) Get user/id
   Возращает имя и аву юзверя
